# Graphtics

Premium GenZ streetwear e-commerce platform built with React, Vite, TailwindCSS, Firebase, and modern frontend architecture.

## Features

- Modern landing experience with motion-first hero, carousel, and trend sections
- Firebase Auth + role-based administration
- Product, cart, wishlist, checkout, and order tracking systems
- Admin analytics dashboard and inventory workflow
- Responsive, accessible, and performance-optimized UI
- SEO meta tags, sitemap guidance, and Firebase Hosting-ready deployment

## Tech Stack

- React 18
- Vite
- TailwindCSS
- React Router
- Framer Motion
- React Query
- Zustand
- Firebase Auth / Firestore / Storage
- React Hook Form + Zod
- Swiper.js
- Headless UI
- Lucide Icons

## Project structure

- `src/` - application source code
- `src/components/` - reusable UI components
- `src/pages/` - route pages and screens
- `src/firebase/` - Firebase configuration and helpers
- `src/store/` - global state management
- `src/hooks/` - reusable hooks
- `src/types/` - shared TypeScript models
- `src/data/` - seed data, product definitions

## Getting started

1. Install dependencies
   ```bash
   npm install
   ```

2. Create Firebase project and add `.env` variables

3. Run development server
   ```bash
   npm run dev
   ```

4. Build for production
   ```bash
   npm run build
   ```

## Deployment

- Configure Firebase Hosting
- Deploy with `firebase deploy`
